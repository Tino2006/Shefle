    brands (trademark registrations)

    │ DELETE: is_admin() ONLY                             │

    ┌─────────────────────────────────────────────────────┐

    │ DELETE: is_admin() ONLY                             │
    └─────────────────────────────────────────────────────┘

    subscription_plans

┌─────────────────────────────────────────────────────────────────────────────┐
│                    ATTACK SCENARIOS PREVENTED                                │
└─────────────────────────────────────────────────────────────────────────────┘

    ❌ Direct Database Query
       → Blocked by RLS (is_admin() returns false)

    ❌ Direct API Call (curl/Postman)
       → Blocked by verifyAdminApi() (403 Forbidden)

    ❌ URL Manipulation (/admin in address bar)
       → Blocked by middleware (redirects to /)

    ❌ Cookie/Session Manipulation
       → Role verified from database on every request

    ❌ Frontend Bypass (disable JS)
       → Server-side checks protect all routes

    ❌ SQL Injection
       → Parameterized queries + RLS

    ✅ RESULT: Multi-layer defense in depth
