## 🚀 USPTO Trademark Search - How to Run (Step 1)

### Prerequisites
✅ Supabase project (already configured)
✅ Node.js and pnpm installed
✅ Next.js app running

---

## Step 1: Get Database Connection String

1. Go to: https://supabase.com/dashboard/project/_/settings/database
2. Find "Connection string" section
3. Select "Transaction pooler" (port 6543)
4. Copy the URI and replace `[YOUR-PASSWORD]` with your database password

---

## Step 2: Add to Environment

Edit `.env.local` and replace `[YOUR-PASSWORD]`:

```bash
DATABASE_URL=postgresql://postgres.rlzeofnciqbmvluksbvl:[YOUR-PASSWORD]@aws-0-us-east-1.pooler.supabase.com:6543/postgres
```

---

## Step 3: Run Database Migration

### Option A: Supabase SQL Editor (Recommended)
1. Go to: https://supabase.com/dashboard/project/_/sql/new
2. Open file: `uspto-trademark-schema.sql`
3. Copy entire contents
4. Paste into SQL Editor
5. Click "Run"

### Option B: Command Line
```bash
psql "[YOUR-DATABASE-URL]" -f uspto-trademark-schema.sql
```

## 🐛 Troubleshooting

### ❌ "DATABASE_URL is not configured"
→ Add DATABASE_URL to .env.local and restart dev server

### ❌ "Unable to connect to database"
→ Check your password is correct in DATABASE_URL
→ Verify using Transaction Pooler URL (port 6543)

### ❌ "relation 'trademarks' does not exist"
→ Run the migration file in Supabase SQL Editor

### ⚠️ Empty results
→ Expected! No data imported yet (Step 1 only)
→ Insert test data (see Step 7, Test 3 above)
